# Axion: AI-Powered Intelligent Investing

Axion is a production-grade, AI-driven intelligent investing platform specifically designed for the Indian market. It bridges the gap between complex market data and actionable insights using cutting-edge Neural Engineering and Predictive Analytics.

![Platform Preview](https://images.unsplash.com/photo-1611974714014-4860013ca63b?auto=format&fit=crop&q=80&w=2070)

## 🚀 Key Features

### 1. Neural Market Signal Overlay
Experience market data like never before. Axion integrates historical performance with **Neural Projections**—AI-predicted price trends visualized directly on interactive charts.
- **Predictive Charts**: See where the stock might go in the next T+0 to T+5 intervals.
- **Forecast Rationale**: Every AI prediction comes with a detailed logical breakdown explaining the "why" behind the signal.
- **Reference Lines**: Clear visual indicators for forecast junction points.

### 2. AI Insights & Sentiment Analysis (Market Pulse)
A real-time "Signal Stream" that synthesizes global market narratives.
- **Granular Sentiment Scoring**: Detailed scoring from -1.0 (Strongly Bearish) to +1.0 (Strongly Bullish).
- **Neural Entity Recon**: Automatically identifies key market players, sectors, and vectors mentioned in news articles.
- **Multi-Source Synthesis**: Aggregates data from verified financial sources to provide a unified market pulse.

### 3. Hyper-Responsive UI/UX
Designed with a "Technical Editorial" aesthetic, the platform is optimized for both precision desktops and mobile-first trading.
- **Captivating Animations**: Powered by `motion/react` for smooth transitions and state-based layout shifts.
- **Dark Mode Architecture**: A depth-first dark UI that minimizes eye strain during long research sessions.
- **Haptic-Feedback Layout**: Touch-optimized controls for mobile users, including quick-preset transaction buttons.

### 4. Smart Transaction System
- **Quick-Adjust Units**: Seamless unit adjustment with haptic presets (+5, +10, +50, +100).
- **Instant Execution Flow**: Streamlined "Capture" (Buy) and "Liquidate" (Sell) workflows.

## 🏁 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn
- A [Google Gemini API Key](https://aistudio.google.com/app/apikey)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/axion-invest.git
   cd axion-invest
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment Variables:**
   Create a `.env` file in the root directory and add your Gemini API key:
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

## 📈 Deployment

The project is configured for seamless deployment. To create a production build:
```bash
npm run build
```
The static files will be generated in the `dist/` directory.
